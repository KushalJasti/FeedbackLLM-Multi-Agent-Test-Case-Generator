Software testing remains a critical yet resource-intensive phase of the software development lifecycle, particularly for programs containing complex control flow, nested conditions, and multiple execution paths. Traditional manual testing and conventional automated techniques often struggle to achieve high coverage efficiently, especially when faced with deep branch constraints or language-specific limitations.

FeedbackLLM addresses these challenges through a multi-agent, iterative test case generation framework powered by large language models. The proposed system consists of a test case generator and two specialized feedback agents: one focused on line coverage gaps and the other on branch coverage gaps. At each iteration, the framework evaluates coverage results, extracts missing execution information, and refines the generation prompt to guide subsequent test creation toward unexplored program paths.

To further enhance practical usability, FeedbackLLM includes a redundancy prevention cache that eliminates duplicate test inputs and reduces redundant API calls. The framework is designed to operate across multiple programming languages, with support for both C and Python, making it a flexible solution for automated test generation and coverage improvement.
